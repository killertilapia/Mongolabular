A short AngularJS app with a Mongolab backend. I could have done this by just using a naked $ngResource adapter but I decided instead to use pkozlowski's mongoLab adapter.

You will need to get a Mongolab account to run this demo for the API key.

I'm using Twitter-Bootstrap 3 for the webpage theme.
